Iris Project

Chat Room

The Iris chat room is a place to meet other Iris users as well as the Iris developers. It is a great place to ask all sorts of questions about Iris. To join the chat room, you will first have to create an account with the MPI-SWS GitLab:

  1. Go to the log-in page and select the "GitHub" button, then log in with your Github account. (Unfortunately, local registration in this GitLab instance is no longer possible due to enormous amounts of spam and inadequate support in GitLab for dealing with such spam.)
  2. Join the Iris chat room.
